# 🍔 Food Menu App (React.js)

A responsive and modern food menu application built using React.js.  
This project displays different food items with category-based filtering and a clean UI design.

---

## 🚀 Features

- 🍽️ Display food menu items dynamically
- 🔍 Category filter (Breakfast, Lunch, Dinner)
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Modern and clean UI
- ⚡ Fast and lightweight React app

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3 (Flexbox + Grid)
- HTML5
